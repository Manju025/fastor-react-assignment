# Fastor App

A React-based web application for discovering and sharing nearby restaurants. This Fastor Assignment demonstrates a full-stack-like experience with user authentication, restaurant listings, and interactive features built using modern web technologies.

## Features

- **User Authentication**: Secure login via phone number and OTP verification (mock OTP: 123456).
- **Restaurant Discovery**: Browse a curated list of nearby restaurants with ratings and addresses.
- **Detailed Views**: View restaurant details with image overlays and sharing capabilities.
- **Responsive Design**: Mobile-friendly interface using Tailwind CSS.
- **Image Sharing**: Generate and share composite images of restaurants with the app logo using HTML5 Canvas.
- **Mock API**: Simulated backend with mock data for restaurants.

## Installation and Setup

### Prerequisites

- Node.js
- npm
- React

### Steps

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd fastor-app
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

### Available Scripts

- `npm run dev` - Start the development server
